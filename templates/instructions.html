Instructions for Using the Freedbody Dial Application



How to Draw!

1) Select the shape you want (line or polygon or ellipse) from the menu.
2) To draw a line, click and then drag the cursor until desired length is achieved.
3) To draw a polygon, click and drag line. When new corner desired click again. Return to start point to close.

How to Edit!

1) Select edit mode from the menu.
2) To resize, select a yellow corner control point and drag.
3) To translate, select the blue square and drag and click accordingly to desired location.

How to Save!

1) Select group mode from the menu.
2) Select the objects you desire to group with your cursor. They will show edit points when selected.
3) Select 'Group Done' when you are done selecting the desired objects.

How to Undo!

1) Select save from the menu.
2) If browser settings are set up so, select location and file name in popup window.

How to Print!

1) Select print mode from the menu.
2) Modify print options in print menu and print.

How to Export!

1) Select export mode from the menu.
2) Select location to export item to.
3) Do not modify this file in anyway - otherwise import will no longer be possible.

How to Import!

1) Select import mode from the menu.
2) Select saved item from location on computer and click "Open" from menu.
